# MegaBlog

MegaBlog is a blog management web application that allows users to create, edit, and manage blog posts with authentication support.

The project focuses on building a clean frontend experience while handling user authentication and real-time data updates.

## Features
- User authentication (login/signup)
- Create, edit, and delete blog posts
- Real-time data updates
- Responsive UI for different screen sizes

## Tech Stack
- React
- Firebase (Authentication & Database)
- Tailwind CSS

## What I Learned
- Managing authentication flow in a frontend application
- Handling CRUD operations with a backend service
- Structuring React components for scalability
- Improving UI responsiveness and user experience
